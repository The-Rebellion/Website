About us

What's a CTF?

CTFs are competitive hacking events in computer security. This is Boot2Root style CTF thus players get a openvpn file to connect to vpn network that will enable them to interact with the vulnerable machine.

The Goal is to hack the vulnerable machine and gain root access. You will be awarded points on submitting the following.

  • User Hash
  • Root Hash

The player will be evalutated on basis of time if two or more players have same points. Moderators decision will be final.
